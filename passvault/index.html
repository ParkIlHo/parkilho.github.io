앱 소개

비번금고는 회원가입 없이 사용하는 100% 로컬 비밀번호 관리 앱입니다. 모든 데이터는 AES 암호화되어 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

주요 기능

  • 사이트별 비밀번호 안전 보관
  • 한국 은행 보안카드 카메라 OCR 자동 입력
  • 은행 계좌 / 신용·체크카드 정보 관리
  • 강력한 비밀번호 생성기
  • 클립보드 자동 삭제 (복사 후 일정 시간 뒤 자동 지움)
  • PIN · Face ID · Touch ID 잠금
  • 백그라운드 전환 시 자동 잠금
  • 민감 화면 스크린샷·녹화 차단

자주 묻는 질문 (FAQ)

Q. 데이터는 어디에 저장되나요?

사용자의 기기 내부에만 AES 암호화되어 저장됩니다. 어떤 서버에도 전송되지 않으며, 개발자도 사용자의 데이터에 접근할 수 없습니다.

Q. 기기를 바꾸면 데이터를 이전할 수 있나요?

현재 버전에서는 백업/복원 기능을 준비 중입니다. 추후 업데이트를 통해 암호화된 파일로 안전하게 내보내고 가져올 수 있도록 지원할 예정입니다.

Q. PIN을 잊어버리면 어떻게 되나요?

PIN은 기기 내부에만 저장되며 복구할 수 없습니다. 잊어버린 경우 앱을 재설치해야 하며, 이 경우 저장된 모든 데이터가 함께 삭제됩니다.

Q. 보안카드 OCR이 잘 안 됩니다.

카드 전체(일련번호 영역 포함)가 프레임 안에 들어오도록 가까이 촬영해주세요. 조명이 충분하고 그림자가 없을 때 인식률이 가장 높습니다. 인식 후에도 미리보기 화면에서 직접 수정할 수 있습니다.

Q. 광고를 보지 않을 수 있나요?

현재 버전은 광고 무료 모델로 운영됩니다. 추후 광고 제거 옵션 도입을 검토 중입니다.

About

PassVault is a 100% local password manager that requires no sign-up. All data is AES-encrypted and stored only on your device — never sent to any server.

Features

  • Save passwords per site / service
  • Camera-based OCR for Korean bank security cards
  • Manage bank accounts and credit / debit cards
  • Strong password generator
  • Auto-clear clipboard after configurable timeout
  • PIN · Face ID · Touch ID lock
  • Auto-lock on background
  • Screenshot / recording protection on sensitive screens

FAQ

Q. Where is my data stored?

Entirely on your device, AES-encrypted. Nothing is sent to any server and even the developer cannot access your data.

Q. Can I migrate data to a new device?

Encrypted backup / restore is in development and will be available in a future update.

Q. What if I forget my PIN?

PINs are stored only on-device and cannot be recovered. You will need to reinstall the app, which will erase all stored data.

Q. Security card OCR is not accurate.

Make sure the entire card including the serial number is in frame, with even lighting and no shadow. You can always edit recognized values manually on the preview screen.

Q. Can I remove ads?

The current version is free with ads. An ad-free option may be added in the future.
